## 项目简介 👋

[Epic AwesomeGamer](https://github.com/QIN2DIM/epic-awesome-gamer) 帮助玩家优雅地领取 Epic 免费游戏。

使用 「Epic免费人」可以实现如下需求：

- `get`：搬空游戏商店，获取所有常驻免费游戏与免费附加内容；
- `claim`：领取周免游戏及其免费附加内容；
- `unreal`：搬空虚幻商店的常驻免费资源，领取本月免费内容；
- `deploy`：部署系统定时任务。
